Featured Artifact

AI Tutor Agent for Algebra 1 Unit 3

The purpose of this artifact was to design an AI tutoring agent that helps Algebra 1 students prepare for their Unit 3 assessment on equations and inequalities. Rather than creating an AI that simply provides answers, I intentionally designed the agent to behave like a tutor by asking guiding questions, diagnosing misconceptions, and encouraging students to explain their thinking. I developed the agent through an iterative process of prompt engineering, testing, and refinement. After creating an initial set of instructions, I tested different scenarios and edge cases to ensure the agent consistently supported student learning without removing the productive struggle that is important in mathematics.

This artifact aligns with Program Objective 3: Use technologies and techniques to design and develop learning materials, media, and environments. By using Microsoft Copilot to develop a specialized AI tutor, I created a technology-enhanced learning environment that extends instruction beyond the classroom. The agent serves as an adaptive learning resource by generating personalized practice, providing immediate feedback, and adjusting its questioning based on student responses. Designing this agent required both technical skills in configuring AI behavior and instructional design decisions that promote meaningful learning, demonstrating how emerging technologies can be intentionally developed to support educational goals.
